# Configuración

Este repositorio incluye archivos de configuración de ejemplo (plantillas) para no subir credenciales reales.

- config/config.example: parámetros del sistema (ESP32 AP, endpoints, tiempos, umbrales).
- config/backend.env.example: conexión a MySQL local (XAMPP) y variables del backend.

Archivos reales (no subir al repositorio):
- config/config.local
- config/backend.env

Estos archivos reales deben crearse copiando las plantillas y reemplazando valores según el entorno.
